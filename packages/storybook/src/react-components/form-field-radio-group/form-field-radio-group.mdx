# Form Field Radio Group

## Gebruik

```tsx
<LuxRadioGroup
  name="contact"
  label="Voorkeur contactmethode"
  options={[
    { value: "email", label: "E-mail" },
    { value: "phone", label: "Telefoon" },
    { value: "mail", label: "Post" },
  ]}
/>
```
