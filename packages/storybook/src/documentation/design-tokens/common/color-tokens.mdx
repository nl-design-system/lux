# Color Design Tokens

Dit zijn de common color tokens, de kleuren die je het meeste zult toepassen. Gebruik de kleuren in de context waar ze voor ontworpen zijn. Gebruik in andere gevallen de [core tokens][core-color-tokens].

## Contrast

Zorg altijd voor een voldoende contrast tussen de achtergrond en de tekst of interactieve elementen. Volg hiervoor de richtlijnen niveau AA van de [Web Content Accessibility Guidelines (WCAG 2.2)][wcag2.2].
