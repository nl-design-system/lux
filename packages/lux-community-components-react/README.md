# Community React Componenten voor het LUX Design System

Deze package bevat de Community React componenten van het LUX Design System.  
Je kan deze package gebruiken in je React- of NextJS-applicatie.

Om dit pakket `@lux-design-system/lux-community-components-react` te gebruiken, heb je waarschijnlijk in je project ook de package `@lux-design-system/lux-community-design-tokens` nodig.

## Installeren

### NPM

`npm install @lux-design-system/lux-community-components-react --save-dev`

### Yarn

`yarn add @lux-design-system/lux-community-components-react --dev`

### PNPM

`pnpm add @lux-design-system/lux-community-components-react --save-dev`
