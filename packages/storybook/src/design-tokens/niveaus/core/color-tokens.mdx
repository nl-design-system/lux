# Color Design Tokens

Gebruik deze kleuren voor ondersteunend materiaal, zoals illustraties, of waar nodig in aangepaste componenten.

Vermijd het gebruik van de core tokens als er een common token is die hiervoor is ontworpen. Bijvoorbeeld, als u de foutstatus van een component ontwerpt, gebruik dan `lux-color-feedback-error` in plaats van `lux-color-rood`.
