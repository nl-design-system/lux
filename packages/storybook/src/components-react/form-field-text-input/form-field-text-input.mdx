## Gebruik

De Lux Form Field Text Input component kan worden gebruikt voor het invoeren van tekst in formulieren. Het ondersteunt verschillende types en staten.

```jsx
import { LuxFormFieldTextInput } from '@lux-design-system/components-react';

<LuxFormFieldTextInput placeholder="Enter your name" />
<LuxFormFieldTextInput type="email" placeholder="Enter your email" />
<LuxFormFieldTextInput type="password" placeholder="Enter your password" />
<LuxFormFieldTextInput type="number" placeholder="Enter a number" />
<LuxFormFieldTextInput required placeholder="Required field" />
<LuxFormFieldTextInput dir="rtl" placeholder="RTL input" />
```
